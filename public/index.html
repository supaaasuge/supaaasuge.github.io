L3AK CTF - Crypto On The Rocks
Posted on 2024-06-01 in CTF • 2484 words • 12 minute read
Tags: Cryptography, ECDSA, Biased Nonces, LLL

Crypto on the Rocks Solution This challenge was inspired by CVE-2024-31497. Within PuTTY, when utilizing the NIST P-521 elliptic curve, the implementation generates nonces with the first 9 bits set to zero. PuTTY’s technique worked by making a SHA-512 hash and then reducing it mod $q$, where $q$ is the order of the group used in the ECDSA system. Introduction This challenge involves breaking the ECDSA (Elliptic Curve Digital Signature Algorithm) using a lattice-based attack.
